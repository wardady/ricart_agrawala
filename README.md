## Install&Run
1. Clone repo:
   ```console
   $ git clone https://github.com/wardady/ricart_agrawala.git
   ```
2. Create virtual environment:
   ```console
   $ python3 -m venv ./env

   $ source env/bin/activate
   ```
3. Install requirements:
   ```console
   $ pip install -r requirements.txt
   ```
4. Run with 3 processes (threads):
   ```console
   $ python3 main.py 3
   ```
